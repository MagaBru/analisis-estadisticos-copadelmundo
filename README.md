## Alumna: Leila Magalí Bruno

## Escenario
Escenario elegido: D - Estadísticas de Resultados Deportivos
Este proyecto tiene como objetivo realizar un análisis estadístico sobre el desempeñp de 
la Selección Argentina de Fútbol a lo largo de las distintas Copas del Mundo de la FIFA.

## Dataset
Los datos utilizados provienen de un archivo de datos que contiene la estadística de partidos,
goles y participaciones históricas del equipo.

## Instrucciones de ejecucuión 
Para realizar el análisis, siga estos pasos:
1. Asegúrese de que el archivo de datos esté ubicado en la carpeta /datos.
2. Ejecute los scripts de código fuente que se encuentran en la carpeta /scripts para procsar info.
3. Una vez realizada la ejecución, los resultados (graficos y tablas) se guardarán automáticamente
   en la carpeta /resultados para su visualización.
